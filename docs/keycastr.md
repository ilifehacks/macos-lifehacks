# KeyCastr
Keystroke visualizer for video recording

- ♻️ free 
- 🌎 [open source](https://github.com/keycastr/keycastr)

## Installing 
[Download latest release](https://github.com/keycastr/keycastr/releases) or via homebrew:

```sh
brew install --cask keycastr
```
